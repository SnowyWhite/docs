This guide walks you through setting up IW4x on Windows using the official launcher, which ensures automatic updates and the smoothest experience.

## Prerequisites

Before you begin, make sure you have a **legitimate copy of Call of Duty: Modern Warfare 2 (2009)** purchased through Steam.

:::caution[Important]
Only the **Steam version** of MW2 is supported. Copies purchased through the Microsoft Store or other platforms will **not work** with IW4x.
:::

## Installation Steps

### 1. Download [Call of Duty: Modern Warfare 2 (2009)](https://store.steampowered.com/app/10180/Call_of_Duty_Modern_Warfare_2_2009/) from Steam
In your Steam library, install **Call of Duty: Modern Warfare 2 (2009) – Multiplayer**. Once that download is finished, you can then continue on.

### 2. Download the official IW4x Launcher
The IW4x launcher is a simple and efficent launcher to handle automatic updates for IW4x. You should use this to launch IW4x to ensure you stay up to date.

### 3. Locate your Modern Warfare 2 folder
In Steam, right-click **Call of Duty: Modern Warfare 2 (2009)**. Select **Manage → Browse local files**. This will open the correct installation folder.

<img src="/src/assets/manage_files.webp"></img>

### 4. Move the IW4x Launcher to your Modern Warfare 2 folder
Move `iw4x-launcher.exe` from your **Downloads** folder into your MW2 game folder.

<img src="/src/assets/drag_iw4x.webp"></img>

### 5. Run the IW4x Launcher
Now open the `iw4x-launcher.exe` inside your game folder. A console window will appear, showing installation progress. Once complete, **IW4x will start automatically.**

:::tip
Always start the game using **iw4x-launcher.exe** or the **IW4x desktop shortcut**.  
This ensures you automatically receive the latest updates.
:::
