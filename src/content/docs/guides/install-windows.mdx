This guide walks you through setting up IW4x on Windows using the official launcher, which ensures automatic updates and the smoothest experience.

## Prerequisites

Before you begin, make sure you have a **legitimate copy of Call of Duty: Modern Warfare 2 (2009)** purchased through Steam.

:::caution[Important]
Only the **Steam version** of MW2 is supported. Copies purchased through the Microsoft Store or other platforms will **not work** with IW4x.
:::

## Installation Steps

### 1. Download [Call of Duty: Modern Warfare 2 (2009)](https://store.steampowered.com/app/10180/Call_of_Duty_Modern_Warfare_2_2009/) from Steam
In your Steam library, install **Call of Duty: Modern Warfare 2 (2009) – Multiplayer**. Once that download is finished, you can then continue on.

### 2. Locate your Modern Warfare 2 folder
In Steam, right-click **Call of Duty: Modern Warfare 2 (2009)**. Select **Manage → Browse local files**. This will open the correct installation folder.

<img src="/src/assets/manage_files.webp"></img>

### 3. Grab the latest IW4x.dll and IW4x.exe
Download both iw4x.dll and iw4x.exe, and make sure you have located the 2 together in your **Downloads**.

If you get a **"Unverified download blocked"** message:  
1. Go to your browser's **Downloads** list.  
2. Click the **three vertical dots** next to the blocked file.  
3. Select **Download unverified file**.  

IW4x is [an open-source client](https://github.com/iw4x/iw4x-client), so the code can be viewed at any time.

The [IW4x rawfiles](https://github.com/iw4x/iw4x-rawfiles) contain modified menus, scripts, and more to make your experience better. This is required to be downloaded. **The download could take a few minutes as its above 700MB of rawfiles.**

### 5. Move the IW4x files to your Modern Warfare 2 folder
Move `iw4x.exe`, `iw4x.dll`, **and the contents of the `release.zip`** from your **Downloads** folder into your Modern Warfare 2 game folder.

Dragging `iw4x.exe` and `iw4x.dll` should look like this:
<img src="/src/assets/drag_iw4x_binaries.webp"></img>

Open up `release.zip`, and then highlight all the files.<br/>
Dragging the contents of `release.zip` should look like this:
<img src="/src/assets/drag_release_zip.webp"></img>

### 6. Run IW4x
Now open the `iw4x.exe` inside your game folder and have fun!
