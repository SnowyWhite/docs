## Изменить настройки Steam

Если IW4x установлена в **такую же** директорию, как и ваша версия Steam, вам нужно будет лишь выполнить следующие действия.  
Если он установлен в **другой** каталог, обратитесь к дополнительным шагам ниже.

<Steps>

1. Откройте Steam и перейдите в библиотеку **LIBRARY**.

2. Щелкните правой кнопкой мыши на **Call of Duty: Modern Warfare 2 (2009) - мультиплеер**

3. Выберите **Свойства...**.

4. На вкладке **Общие** найдите раздел **Язык**.

5. Нажмите на выпадающее меню и выберите нужный язык

</Steps>

После смены языка Steam загрузит необходимые файлы для выбранного варианта.  
После завершения загрузки вы можете запустить IW4x и наслаждаться игрой в удобных для вас настройках!

<Video src={change_language} />

:::caution

Обратите внимание, что загрузка новых языковых файлов может занять некоторое время, в зависимости от вашего интернет-соединения.  
Видео, представленное выше, было ускорено в демонстрационных целях.

:::

## Другое место установки

Если IW4x установлена отдельно от вашей версии Steam, вам придется вручную скопировать языковые файлы.  
Инструкции по их получению см. в [разделе выше](#change-steam-settings).

### Файлы для копирования

Убедитесь, что вы скопировали следующие файлы и папки из вашей установки Steam:

- Файл `localization.txt`
- Папка `main/video/`
- Папка `zone/<language>/` (e.g., `zone/russian/` for Russian)
- Все файлы с именем `localized_<язык>_iw##.iwd` в папке `main/` (например, `localized_russian_iw00.iwd` для русского языка)

Обзор структуры файлов ####

<FileTree>
- `Call of Duty: Modern Warfare 2`
  - main
    - **video**
      - *.bik all files
    - **localized_\<язык\>_iw00.iwd** 
    - **..** All `.iwd` файлы для выбранного языка
    - **localized_\<язык\>_iw##.iwd**
  - zone
    - **\<язык\>/**
      - *.ff все файлы
  - **localization.txt**
</FileTree>

### Файлы для удаления

Теперь в вашей папке `main` находятся файлы `localized_<язык>_iw##.iwd` как для **вновь выбранного языка, так и для того, который вы использовали ранее.
Если вы запустите игру, вы увидите ошибку, подобную этой:

![Couldn't load image 'specialty_new_nonenglish'](../../../assets/img/language/error_message.png)

Чтобы исправить это, вам нужно будет выполнить действия с файлами из **предыдущего языка**.  
Вы можете выбрать один из следующих вариантов:

- **Переименуйте их** - Например, добавьте `.bak` в конец: `localized_english_iw00.iwd` -> `localized_english_iw00.iwd.bak`
- **Переместите их** - например, создайте папку `english` рядом с `main` и переместите туда файлы.
- **Удалите их полностью**.

Какой метод использовать - решать вам. Главное, чтобы папка `main` содержала файлы только для **одного языка**.

### Пример: Русский язык

Если вы хотите добавить русский язык в качестве нового языка, скопируйте следующие файлы из вашей версии Steam в папку IW4x по тем же относительным путям:

<FileTree>
- `Call of Duty: Modern Warfare 2`
  - main
    - **video/**
    - **localized_russian_iw00.iwd**
    - **localized_russian_iw01.iwd**
    - **localized_russian_iw02.iwd**
    - **localized_russian_iw03.iwd**
    - **localized_russian_iw04.iwd**
    - **localized_russian_iw05.iwd**
    - **localized_russian_iw06.iwd**
    - **localized_russian_iw07.iwd**
    - **localized_russian_iw08.iwd**
  - zone
    - **russian/**
  - **localization.txt**
</FileTree>
