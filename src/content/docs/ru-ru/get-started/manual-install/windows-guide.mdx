:::caution

Мы настоятельно **рекомендуем** использовать [IW4x Launcher](https://github.com/iw4x/launcher), из [Quickstart Guide](/get-started/quickstart).  
Это самый простой и надежный способ установить и поддерживать IW4x в актуальном состоянии.  
Однако если вы предпочитаете устанавливать IW4x вручную, вы можете выполнить следующие шаги.  
**Важно**: Если вы устанавливаете IW4x вручную, вам придется повторять все эти шаги каждый раз, когда выходит новое обновление.

:::

## 1. Найдите папку с установленной игрой Modern Warfare 2

Iw4x должен быть установлен в тот же каталог, в котором находится игра *Call of Duty: Modern Warfare 2*.  
Обычно путь к этому каталогу следующий:

```cmd
C:\Program Files (x86)\Steam\steamapps\common\Call of Duty Modern Warfare 2
```

Если у вас возникают проблемы с поиском каталога установки, вы можете воспользоваться своей библиотекой Steam в качестве помощи:

<Steps>

1. Откройте Steam и перейдите в свою библиотеу **LIBRARY**

2. Щелкните правой кнопкой мыши на игре **Call of Duty: Modern Warfare 2 (2009) - Мультиплеер**

3. Выберите **Управление > Просмотр локальных файлов**

    <Video src={find_directory} />

</Steps>

## 3. Скачайте Raw файлы

[Raw-файлы](https://github.com/iw4x/iw4x-rawfiles) - это дополнительные ресурсы, такие как карты, оружие и скрипты, которые улучшат ваш опыт работы с IW4x.  
Как и на [предыдущем шаге](#2-download-iw4x), загрузите и сохраните последнюю версию Raw-файлов в каталог вашей игры.

<LinkButton
    href="https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x.exe"
    variant="primary"
    icon="seti:windows"
    iconPlacement="start"
>Скачайте **iw4x.exe**</LinkButton>

<LinkButton
    href="https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/release.zip"
    variant="primary"
    icon="seti:windows"
    iconPlacement="start"
>Скачайте **release.zip**</LinkButton>

На GitHub вы увидите еще несколько загрузок, но нас интересуют:

- **iw4x.exe** - Пользовательская версия оригинального `iw4mp.exe", предназначенная для запуска `iw4x.dll" (скачено на предыдущем шаге).
- **release.zip ** - Содержит все необходимые дополнительные ресурсы для IW4x

Дополнительные файлы, с которыми вы можете столкнуться:

- `zonebuilder.exe` - Инструмент для создания пользовательских ресурсов и модов
- "update.json" - Используется лаунчером для отслеживания обновлений, может быть проигнорирован
- Несколько файлов с расширением `.iwd` (например, `iw4x_00.iwd`)

Файлы `.iwd` содержат дополнительный контент для DLC, включая Nuketown из *Call of Duty: Black Ops (2010)* и классические карты из *Call of Duty: Modern Warfare (2007)*.  

IW4x будет работать и без них, но вы должны их скачать, если хотите играть на DLC картах.  
Не забудьте переместить их в каталог "iw4x/", иначе игра их не распознает.

Каталог `iw4x/` еще не существует, это потому, что сначала необходимо извлечь файлы из **release.zip**.  
Самый простой способ сделать это - выбрать **release.zip**, а затем:

<Steps>

1. Правый клик **Открыть с помощью - Проводник**

2. Нажмите на **Извлечь все**

3. Нажмите на **Извлечь**

Файл release.zip должен быть извлечен в корневой каталог вашей игры папки (не в дополнительную подпадку `Release`, а в корень игры).

4. Удалите файл **release.zip**

</Steps>

## Скачивание и установка с помощью PowerShell

:::note[Info]
Вы можете пропустить этот раздел, если вы уже выполнили предыдущие шаги или если вы не знакомы с терминалом.  
В приведенных ниже инструкциях просто показано, как выполнить те же действия, но с помощью PowerShell.
:::

<Steps>

1. **`переход` в [каталог игры](#1-find-modern-warfare-2-installation)**

   ```ps
   cd "C:\Program Files (x86)\Steam\steamapps\common\Call of Duty Modern Warfare 2"
   ```

2. **Загрузка `iw4x.dll`**

   ```ps
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-client/releases/latest/download/iw4x.dll
   ```

3. **Загрузка `iw4x.exe`**

   ```ps
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x.exe
   ```

4. **Загрузка `release.zip`**

   ```ps
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/release.zip
   ```

5. **Распаковка `release.zip`**

   ```ps
   Expand-Archive release.zip -DestinationPath .
   ```

6. **Удалите `release.zip`**

   ```ps
   Remove-Item release.zip
   ```

7. **Загрузка `.iwd files`**

   ```ps
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_00.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_00.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_01.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_02.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_03.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_04.iwd iw4x
   Start-BitsTransfer -source https://github.com/iw4x/iw4x-rawfiles/releases/latest/download/iw4x_05.iwd iw4x
   ```

</Steps>
